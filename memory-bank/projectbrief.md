# 프로젝트 개요 (Project Brief)

## 프로젝트 정보
- **프로젝트명**: hello-nextjs
- **버전**: 0.1.0
- **타입**: Next.js 웹 애플리케이션

## 핵심 목표
- Next.js 15.5.0 기반의 현대적인 웹 애플리케이션 개발
- React 19.1.0과 TypeScript를 활용한 타입 안전한 개발
- Tailwind CSS를 이용한 현대적인 UI 구현

## 프로젝트 범위
- 기본적인 Next.js App Router 구조
- 반응형 웹 디자인
- 최신 React 기능 활용
- 성능 최적화 (Turbopack 활용)

## 주요 요구사항
- 사용자 친화적인 인터페이스
- 빠른 로딩 속도
- SEO 최적화
- 접근성 준수

## 성공 지표
- 페이지 로드 속도 < 2초
- 모바일 친화적 디자인
- 웹 접근성 AA 등급 준수
